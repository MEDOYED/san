Mapa Witryny

Ta strona zawiera pełną strukturę nawigacji portfolio Maksyma Mokriakova.

Poniżej znajdziesz hierarchiczną listę wszystkich dostępnych stron wraz z ich opisami.

  • Strona Główna (index.html) Strona tytułowa zawierająca dane osobowe, zdjęcie autora i szybkie linki do wszystkich podstron portfolio
    • O mnie (about.html) Informacje o autorze, jego pasjach, zainteresowaniach programowaniem i technologiami webowymi
    • Edukacja (education.html) Ścieżka edukacyjna: kolegium technologiczne w Łomży, studia informatyczne na SAN, rozwój kompetencji technicznych
    • Hobby (hobby.html) Zainteresowania poza programowaniem: łucznictwo na dystansach 30 i 60 metrów, regularne treningi siłowe, równowaga między pracą umysłową a fizyczną
    • Umiejętności (skills.html) Technologie i narzędzia: języki programowania (JavaScript, Python, C++), frameworki (React, Node.js), bazy danych, systemy kontroli wersji
    • Projekty (projects.html) Zrealizowane projekty: UniPrint (system zarządzania drukowaniem), Prospi.ai (platforma AI), osiągnięcia i plany rozwoju zawodowego
    • Kontakt (contact.html) Dane kontaktowe: email, telefon, linki do profili GitHub i LinkedIn, możliwości współpracy
    • Mapa Witryny (sitemap.html) Ta strona - pełna hierarchiczna struktura nawigacji portfolio

Dodatkowe Informacje

Technologie użyte w witrynie:

  • HTML5 - semantyczny markup zgodny ze standardami W3C
  • CSS3 - nowoczesne stylowanie z CSS Grid i Flexbox
  • JavaScript ES6+ - interaktywność i responsywne menu
  • WCAG 2.1 - zgodność ze standardami dostępności cyfrowej

Funkcje dostępności:

  • Nawigacja klawiaturowa (Tab, Enter, Escape)
  • Skip link do głównej treści
  • ARIA atrybuty dla technologii asystujących
  • Teksty alternatywne dla wszystkich obrazów
  • Opisowe podpisy figcaption dla lepszego kontekstu
  • Kontrast kolorów zgodny z WCAG AA (min. 4.5:1)
  • Responsywny design dla wszystkich urządzeń
  • Semantyczny HTML5 (header, nav, main, article, footer)
